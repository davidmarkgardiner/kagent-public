# Team Handover Pack

## TL;DR

Human-facing launch pack for Kagent triage v2: GitLab tickets, Teams messages,
game-day operating plan, and stakeholder/SRE presentation.

## What This Feature Does

- Gives the team copyable tickets and messages.
- Frames the SRE adoption and feedback loop.
- Provides a game-day operating plan.
- Provides an HTML presentation for walkthroughs.
- Points to the technical bundles for implementation proof.

## Evidence To Produce

- Selected GitLab tickets.
- Teams launch message.
- Game-day schedule or owners.
- Presentation reviewed.
- Feedback loop and improvement routing agreed.

## How To Run

1. Run `bash scripts/verify-bundle.sh`.
2. Review `GITLAB-TICKETS.md`.
3. Review `TEAMS-MESSAGES.md`.
4. Review `GAMEDAY-OPERATING-PLAN.md`.
5. Open `presentation/kagent-triage-v2-sre-handover.html`.

## Definition Of Done

The team can brief stakeholders, create work items, schedule SRE sessions, and
route feedback into the technical bundle backlog.
