Grafana Kafka Alert Verification

Prove real alert firing, consume the Kafka event, capture the payload, and make the schema decision from evidence.

1. PreflightCheck variables, Grafana MCP, Kafka tooling, and access.
2. FireCreate or use a temporary always-firing Grafana rule.
3. ConsumeRead the produced Kafka record with metadata.
4. ValidateCompare the captured payload to the draft schema.
5. DecideConsumer-side schema now, bridge/serializer for broker validation.

Evidence Markers

  • ENV_PREFLIGHT
  • GRAFANA_CONTACT_POINT
  • ALERT_RULE
  • KAFKA_RECORD
  • PAYLOAD_CAPTURED
  • SCHEMA_VALIDATION
  • CLUSTER_CONSUMER
  • BROKER_SCHEMA_DECISION
