# Reference

Use the existing smart-triage GitLab MCP proof as the local pattern:

- `SMART-TRIAGE-GITLAB-MCP-MR-DEMO.md`
- `a2a/smart-triage-fanout-demo/gitlab-mcp-remotemcpserver.yaml`
- `a2a/smart-triage-fanout-demo/gitlab-lite-mcp.yaml`
- `a2a/smart-triage-fanout-demo/scripts/run-gitlab-mr-demo.sh`

At work, prefer the official or already-installed GitLab MCP path if it exposes
branch, file, merge request, and note tools. Use an approved wrapper only if the
installed server lacks the required write tools.
