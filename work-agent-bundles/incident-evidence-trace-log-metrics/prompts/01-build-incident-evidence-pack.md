# Prompt: Build Incident Evidence Pack

```text
Build a concise incident evidence pack using Grafana MCP read tools.

Return:
- metrics datasource and PromQL result;
- logs datasource and LogQL result;
- trace lookup result or explicit NO_TRACE fallback;
- dashboard or panel link;
- source queries and timestamps;
- triage synthesis update;
- proof that the evidence agent has no mutation tools;
- blockers and next actions.
```
