# GitLab Ticket: Start SRE Adoption Feedback Loop

## Summary

Run the first SRE adoption loop for one application and capture whether the
Kagent triage v2 workflow is usable in real SRE day-to-day work.

## Feature

SRE should select an application, use the kagent workflow during a first-contact
or game-day session, provide feedback, and see improvements routed to owners.

## Evidence Required

- SRE owner and application.
- First-contact run.
- Controlled exercise result.
- Workflow usage evidence.
- Feedback captured.
- Improvement item routed.
- Adoption report.

## Acceptance Criteria

- `SRE_OWNER_IDENTIFIED: yes`
- `APPLICATION_SELECTED: yes`
- `FIRST_CONTACT_RUN_COMPLETED: yes`
- `CHAOS_OR_INCIDENT_EXERCISE_COMPLETED: yes`
- `KAGENT_WORKFLOW_USED_BY_SRE: yes`
- `FEEDBACK_CAPTURED: yes`
- `IMPROVEMENT_ITEM_ROUTED: yes`
- `ADOPTION_REPORT_CREATED: yes`
- `OUTPUT_SANITIZED: yes`

## Notes

This is the operating loop. It should use the technical bundles rather than
replacing them.
